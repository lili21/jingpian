# 镜片 Professional Landing Page

静态落地页实现文件：

- `index.html`
- `styles.css`
- `script.js`

## 本地预览

直接在浏览器中打开 `index.html` 即可。

如果你希望使用本地静态服务（可选），可在项目目录执行：

```bash
python3 -m http.server 8080
```

然后访问：<http://localhost:8080>

## 表单说明

页面底部表单提交后会将数据保存到浏览器 `localStorage`，键名为：

`jingpian_professional_applications`

用于演示提交成功状态与前端存储流程。
